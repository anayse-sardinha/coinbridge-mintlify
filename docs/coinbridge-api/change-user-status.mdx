---
title: Change User Status
description: "Suspend or unsuspend a user's CoinBridge account."
api: "POST /api/v1/users/{userId}/status"
---

Suspends or unsuspends a user. A suspended user's card is blocked immediately and they cannot make payments until you send an `UNSUSPEND` request.

**`POST /api/v1/users/{userId}/status`**

## Path Parameters

<ParamField path="userId" type="string" required>
  The unique identifier of the user.
</ParamField>

## Request Body

<ParamField body="status" type="string" required>
  The action to apply. One of: `SUSPEND` or `UNSUSPEND`.
</ParamField>

<ResponseExample>
```json Request
{
  "status": "SUSPEND"
}
```
</ResponseExample>
