This guide walks you through every step required to go from a fresh install to live payments.

## Card Setup

<Steps>
  <Step title="Prerequisites">
    Generate your RSA key pair, add the SDK to your project, and configure the required permissions before writing any integration code.

    [Get started](/docs/sdk/to-start)
  </Step>

  <Step title="Authenticate">
    Call `authenticate()` with a JWT generated by your backend to verify the user's identity. Only needed if the user is not already authenticated.

    [Authentication guide](/docs/sdk/installation/authentication)
  </Step>

  <Step title="Initialize">
    Call `initialize()` on every app launch to sync the SDK with the CoinBridge backend. Returns a `hasCard` boolean that tells you what to do next.

    [Initialization guide](/docs/sdk/installation/initialize)
  </Step>

  <Step title="Issue a Digital Card">
    If `hasCard` is `false`, call `issueCard()` to create the user's permanent digital card.

    [Card issuance guide](/docs/sdk/installation/issue-card)
  </Step>

  <Step title="Add to Wallet">
    Call `provisionCard()` on Android or `addToApplePay()` on iOS to push the card token to the device wallet. Once the token status is `ACTIVE`, the user can tap and pay.

    [Add to wallet guide](/docs/sdk/installation/add-to-wallet)
  </Step>
</Steps>
