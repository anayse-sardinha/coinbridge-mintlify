## How the CoinBridge Payment Process Works
Access a detailed step-by-step guide of the entire process and links to dedicated technical pages.

### Set Up a Card
Follow these steps to set up your card:

<CardGroup cols={2}>
  <Card title="1. Install" icon="download">
    Import the SDK and configure **NFC** and **Biometrics** permissions.
  </Card>
  
  <Card title="2. Authenticate" icon="lock">
    Authenticate users with a **JWT token** signed by your backend.
  </Card>
  
  <Card title="3. Initialize" icon="gears">
    Start the SDK by **syncing with the CoinBridge backend** on app launch.
  </Card>
  
  <Card title="4. Create a Virtual Card" icon="credit-card">
    Use the `IssueCard` method to create a virtual card for the authenticated user.
  </Card>
  
  <Card title="5. Add Card to Wallet" icon="wallet">
    Push the digital token to **Apple or Android Wallet** for NFC payments.
  </Card>
</CardGroup>
