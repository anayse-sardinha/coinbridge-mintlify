## Setup a Card
Follow these steps to set up your card:

<Columns cols={2}>
  <Card title="1. Prerequisite" icon="list-check" href="/android-sdk/get-started/to-start">
    Set up your RSA keys and repository credentials before importing the SDK.
  </Card>

  <Card title="2. Authenticate" icon="key" href="/android-sdk/sdk/installation/authentication">
    Authenticate users with a JWT token signed by your backend.
  </Card>

  <Card title="3. Initialize" icon="arrows-rotate" href="/android-sdk/sdk/installation/initialize">
    Start the SDK by syncing with the CoinBridge backend on app launch.
  </Card>

  <Card title="4. Issue a Digital Card" icon="credit-card" href="/android-sdk/sdk/installation/issue-card">
    Use the `IssueCard` method to create the user's permanent digital card.
  </Card>

  <Card title="5. Add Card to Wallet" icon="wallet" href="/android-sdk/sdk/installation/add-to-wallet">
    Push the digital token to Android Wallet for NFC payments.
  </Card>
</Columns>

<Note>
  Steps 1, 2, 4, and 5 are typically one-time actions performed during a user's initial onboarding. Step 3 (Initialization) is required every time the application launches.
</Note>
