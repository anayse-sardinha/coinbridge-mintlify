title: "Before You Start"
---

To use the CoinBridge bridge infrastructure, you must deploy the SDK into your mobile application. Follow these steps to configure your environment and add the SDK for Android (Gradle).

Before starting the integration, ensure your environment meets the following requirements:

* **OS Version**: Android 8.0 Oreo (API level 26) or higher.
* **Tooling**: Use Android API level 30 for compilation.
* **Device**: Must be NFC-equipped and not rooted.

## 1. Configure Credentials
Add a `coinbridge.properties` file in the root folder of your project. Add the following lines using the credentials provided by the CoinBridge team:

```
cbpr.url=https://maven.pkg.github.com/CoinBridgeGit/AndroidSDKRelease
cbpr.name=GitHubPackages
cbpr.user=<your-username>
cbpr.password=<your-password>
```
<Note> For the Sandbox environment, change ```AndroidSDKRelease``` to ```AndroidSDKSandbox```. </Note>

<Warning> **Important:** Do not commit this file to source control as it contains private credentials </Warning>

## 2. Add Maven Repositories
In your project-level `build.gradle file`, add the following repository configuration:

```
allprojects {
  repositories {
    //...
    maven { url 'https://jitpack.io' }
    maven {
      def cbProperties = new Properties()
      cbProperties.load(new FileInputStream(rootProject.file("coinbridge.properties")))

      name = cbProperties[`cbpr.name`]
      url = uri(cbProperties[`cbpr.url`])
      credentials {
        username = cbProperties[`cbpr.user`]
        password = cbProperties[`cbpr.password`]
      }
    }
  }
}
```

## 3. Add Module Dependencies
In your module's `build.gradle file`, add the SDK declaration and mandatory native library protection.

```
android {
  packagingOptions {
    // Required to prevent integrity check failures for libscm.so
    doNotStrip "*/*/libscm.so" 
    
    // Required if minSdkVersion is 23+ or using Gradle 8+
    jniLibs {
      useLegacyPackaging = true 
    }
  }
}

dependencies {
  // Replace x.y.z with the required version
  implementation 'com.coinbridge:cbsdk:x.y.z' 
}
```

<Note> For the Sandbox environment, change `cbsdk` to `cbsdk-sb`. </Note>

### 4. App Bundle Configuration
If you are building your application as an Android App Bundle, you must set the following in your `gradle.properties` file to ensure native library integrity checks pass:

```
android.bundle.enableUncompressedNativeLibs=false 
```
