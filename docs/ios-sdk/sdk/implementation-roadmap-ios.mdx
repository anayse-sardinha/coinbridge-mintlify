## Set Up a Card
Follow these steps to set up your card:

<Columns cols={2}>
  <Card title="1. Install" icon="list-check" href="/ios-sdk/get-started/to-start">
    Import the SDK and configure NFC and Biometrics permissions.
  </Card>

  <Card title="2. Authenticate" icon="key" href="/ios-sdk/sdk/installation/authentication">
    Authenticate users with a JWT token signed by your backend.
  </Card>

  <Card title="3. Initialize" icon="arrows-rotate" href="/ios-sdk/sdk/installation/initialize">
    Start the SDK by syncing with the CoinBridge backend on app launch.
  </Card>

  <Card title="4. Issue a Digital Card" icon="credit-card" href="/ios-sdk/sdk/installation/issue-card">
    Use the `IssueCard` method to create a virtual card for the authenticated user.
  </Card>

  <Card title="5. Add Card to Wallet" icon="wallet" href="/ios-sdk/sdk/installation/add-to-wallet">
    Push the digital token to Apple or Android Wallet for NFC payments.
  </Card>
